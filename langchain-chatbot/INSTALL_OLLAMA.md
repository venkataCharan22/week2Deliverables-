# Quick Ollama Installation for Mac

## Option 1: Direct Download (Recommended)
1. Download from: https://ollama.com/download
2. Open the downloaded .dmg file
3. Drag Ollama to Applications
4. Open Ollama from Applications

## Option 2: Terminal Installation
Run this command and enter your password when prompted:
```bash
curl -fsSL https://ollama.com/install.sh | sh
```

## After Installation
1. Ollama should start automatically (check menu bar)
2. Or start it manually: `ollama serve`
3. Pull a model: `ollama pull llama3`
4. Verify: `ollama list`

Then return to the main README.md to run the chatbot!
