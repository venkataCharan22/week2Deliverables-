## Projects

### 1. RAG Chatbot with Vector Database

A Retrieval-Augmented Generation (RAG) chatbot using **ChromaDB** as a vector database.  
Upload PDF, TXT, or MD documents, process them into vector embeddings, and ask questions based on the uploaded content.  
Built with **LangChain**, **Ollama**, and **Streamlit**.

### 2. LangChain Chatbot with Ollama

A conversational AI chatbot built with **LangChain**, **Ollama**, and **Streamlit**.  
Supports multiple local LLM models (llama3, mistral, llama3.2, gemma2) with conversation memory that remembers context across messages.
